Декоративные изображения

  1. CSS background-image
  2. пустой alt=""
  3. role="presentation"
  4. aria-hidden="true":
                <!--Плохо-->
                 < img src="gradient.png"  >  
                alt="Градиент на фоне текста" 
                 
                 <!--Хорошо-->
                 < img src="gradient.png"  
                 aria-hidden="true" alt="" >

Графики, диаграммы

  1. краткое описание в атрибут alt
  2. полное описание на отдельную страницу в тег figcaption
  3. атрибуты longdesc или aria-describedby
 
           < img src="bar-chart.png"   >  
          alt="График посещений сайта weblind.ru за 2016 год"  
           < a href="chart-info.html">Описание изображения </a >   
        
 
           < figure role="group"  >  
           < img src="chart-bar.png"  >  
           alt="График посещений сайта weblind.ru за 2016 год"  
           <figcaption>  
           < a href="chart-info.html">График посещений сайта weblind.ru за 2016 год </a >   
           <figcaption"> 
        
 
           < img src="chart-bar.png"  >  
           alt="График посещений сайта weblind.ru за 2016 год"  
           longdesc="#chart-longdesc"  
           <p> id="chart-longdesc">График посещений сайта  
            weblind.ru за 2016 год ... </p>

Управлять звуком и видео

  1. Альтернативное описание видео и аудио
  2. aria-describedby, longdesc
  3. Невидимый для обычного пользователя блок с информацией.
  4. кнопки для остановки, воспроизведения и регулировки громкости звуков независимо от громкости системы.

Формы ввода

  1. Label
  2. Title
  3. aria-label
  4. aria-describedby
  5. aria-invalid
  6. aria-required
a11y-guidelines.orange.com

Поля ввода

      <!--Плохо-->
       <label > Электронная почта </label >
       <input type="text">   
       
       <!--Хорошо-->
       <label for="input-mail"> Электронная почта </label >
       <input type="text"  aria-invalid="false"    
         aria-required="true" for="input-mail" >

Если label нужно скрыть

           <label for="input-mail"> Электронная почта </label >
           <input type="text" id="input-mail">   
        
 
       .visually-hidden {
          position: absolute;
          width: 1px;
          height: 1px;
          margin: -1px;
          border: 0;
          padding: 0;
          clip: rect(0 0 0 0);
          overflow: hidden;
       } 
      
              <!--Плохо-->
                display: none
                visibility: hidden.
